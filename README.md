# Student Grade Management System

A simple **Windows Forms** application built in C# for managing student names and grades. Users can add students, view all students, search for a student, calculate the average grade, and find the highest and lowest grades.

---

## Features

- Add a student with name and grade
- Display all students in a list
- Search for a student by name
- Calculate average grade
- Find highest and lowest grades
- User-friendly GUI using Windows Forms

---

## Technologies

- C#
- .NET Framework
- Windows Forms (WinForms)
